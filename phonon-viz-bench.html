Browse theme packs from the ☰ menu (top-left) — one at a time, each fed the same VizState snapshot at ~60 fps. Drive the controls on the left to watch the active pack react to inference, battery drain, heat, and cluster layout. Low-power mode engages automatically at ≤20% battery off charger.

Parity build. Every pack here mirrors the Kotlin Render() drawing logic one-to-one — same primitives, same state mapping, same low-power degradation — so anything tuned on this bench ports straight back to the sidecar APK.
